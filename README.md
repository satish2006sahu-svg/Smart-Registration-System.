# User Profile Validation System

## Description
The Smart Registration System is a Python-based credential validation program designed to simulate a university registration approval process.
It validates user-entered Student ID, Email ID, Password, and Referral Code based on predefined university rules using basic Python concepts only.

## Validation Rules
- Student ID must follow the format CSE-XXX.
- Email ID must contain @ and . and must end with .edu.
- Password must be at least 8 characters long, start with an uppercase letter, and contain at least one digit.
- Referral code must follow the format REF##@.

## Constraints Followed
- No loops.
- No advanced inbuilt functions.
- No regular expressions.
- No advanced libraries.
- Use only data types, strings, and conditional statements.
## Strict Constraints
- No loops.
- No regex.
- No lists/dicts.
- No advanced built-ins.
- Only strings + conditions.

## Language Used
Python
